Exercício: Algoritmo para Limpar um Cômodo da Casa

Neste exercício, o objetivo é criar um algoritmo que ajude o usuário a organizar a limpeza de um cômodo da casa. O programa deve permitir que o usuário escolha entre três cômodos (cozinha, quarto ou banheiro), mostrar os materiais necessários para a limpeza e os passos recomendados para cada cômodo. Ao final, o usuário pode decidir se quer limpar outro cômodo ou encerrar a tarefa.

Como funciona o algoritmo:

  • O usuário escolhe qual cômodo deseja limpar: cozinha, quarto ou banheiro.
  • O programa mostra uma lista de materiais necessários para aquele cômodo.
  • O programa exibe um passo a passo de como limpar o cômodo escolhido.
  • Após finalizar, o usuário pode escolher limpar outro cômodo ou encerrar.
  • Quando todos os cômodos forem limpos, o programa parabeniza o usuário.

Exemplo de código em Python:


materiais_quarto = ["Vassoura", "Pano", "Espanador", "Saco de lixo"]
materiais_cozinha = ["Vassoura", "Pano", "Detergente", "Esponja", "Saco de Lixo"]
materiais_banheiro = ["Desinfetante", "Escova", "Pano", "Balde", "Saco de Lixo"]
comodos_limpos = []

while len(comodos_limpos) < 3:
    opcoes = []
    if "cozinha" not in comodos_limpos:
        opcoes.append("1. Cozinha")
    if "quarto" not in comodos_limpos:
        opcoes.append("2. Quarto")
    if "banheiro" not in comodos_limpos:
        opcoes.append("3. Banheiro")
    print("\nEscolha qual cômodo você quer limpar:")
    print('\n'.join(opcoes))
    escolha = int(input("Digite sua Resposta: "))

    if escolha == 1 and "cozinha" not in comodos_limpos:
        print(f"Pegue esses materiais: {', '.join(materiais_cozinha)}")
        print("1. Lave a louça\n2. Limpe a pia\n3. Varra a cozinha\n4. Passe pano no chão\n5. Tire o lixo")
        comodos_limpos.append("cozinha")
    elif escolha == 2 and "quarto" not in comodos_limpos:
        print(f"Pegue esses materiais: {', '.join(materiais_quarto)}")
        print("1. Arrume a cama\n2. Tire o lixo\n3. Varra o quarto\n4. Passe pano\n5. Limpe os móveis")
        comodos_limpos.append("quarto")
    elif escolha == 3 and "banheiro" not in comodos_limpos:
        print(f"Pegue esses materiais: {', '.join(materiais_banheiro)}")
        print("1. Lave o vaso\n2. Limpe a pia\n3. Varra o banheiro\n4. Passe pano\n5. Tire o lixo")
        comodos_limpos.append("banheiro")
    else:
        print("Opção inválida ou cômodo já limpo. Tente novamente.")
        continue

    if len(comodos_limpos) < 3:
        continuar = input("Deseja limpar outro cômodo? (s/n): ")
        if continuar.lower() != 's':
            certeza = input("Tem certeza que não quer limpar os outros cômodos? (s/n): ")
            if certeza.lower() == 's':
                print("Ok! Se mudar de ideia, volte para deixar tudo brilhando! :)")
                break
            else:
                continue

if len(comodos_limpos) == 3:
    print("Uau! Você limpou todos os cômodos da casa! Agora sim, pode descansar e curtir um ambiente limpinho. Parabéns pelo esforço!")

Resumo:

Esse exercício ajuda a praticar lógica de programação, uso de listas, laços e condicionais em Python, além de mostrar como automatizar tarefas do dia a dia!
